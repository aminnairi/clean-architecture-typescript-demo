# clean-architecture-weblab

Clean Architecture implementation using TypeScript.

## Usage

### Console

```bash
make console
```

### Web

```bash
make web
```

## Installation

```bash
git clone https://github.com/aminnairi/clean-architecture-typescript-demo.git
cd clean-architecture-typescript-demo
```

## Requirements

- Git
- Docker
- Docker Compose
- Bash
- GNU/Make
